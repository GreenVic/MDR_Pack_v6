Exe/TextInsert.exe - вставляет текст вместо ключей, начинающихся с #.

============= 1 - Сборка описаний блоков периферии ========
BKP, EEPROM, RST, GPIO, ... - блоки описания периферии.

  На примере BKP:
    - BKP_Common.xml - Бланк, содержит общие части описания и тегов для вставки
    - BKP_Text.txt - Текст, который вставляется вместо тагов. Может быть несколько.
    - TAGS_VEхх.txt - Зависимости. В каком файле лежит текст для таков под конкретный МК.

  Run.bat - сборка описания блока, который будет вставляться в общий svd файл микроконтроллера.
    - BKP_VE1.txt и т.д.

============= 2 - Сборка svd под Микроконтроллер ========
_src - СБОРКА SVD под МК.
  Аналогично сборке блоков.

  Update.bat - обновление сборки отдельных блоков. Запуск Run.bat в папаках BKP, EEPROM, RST, GPIO, ... и т.д.
  Run.bat - Сбока конечных svd.

============= 3 - Проверка svd и создание Headers ========
_out - Директория для данного шага.

  makeAll.bat - проверка svd, лог в файлах типа _out\out\logVE1.txt - должно быть без ошибок.



====================== Исправление ошибок, пересборка =================
1 - Исправить ошибки в текстах блоков. 

2 - Пересобрать тексты блоков с исправлениями
  А - если ошибка одна - перекомпилировать один блок, например вызвать	: BKP/Run.bat.
  Б - если ошибок несколько - полная перекомпиляция всех блоков		: _src/Update.bat

3 - Сборка svd:   _src/Run.bat
4 - Проверка:     _out\makeAll.bat
