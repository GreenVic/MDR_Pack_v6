Настроки в MDR_Config.h необходимо сконфигурировать в соответствии с настройками на конкретной плате

Devices - Файлы описания конкретного микроконтроллера

Drivers - Исходники работы с периферийными модулями, общие для всех МК

Boards - Определения для отладочных плат

ICs  - Функции для работы микроконтроллера с некоторыми микросхемами.
